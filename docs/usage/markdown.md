# Markdown Grammar

## Title

```markup
# h1
## h2
### h3
#### h4
##### h5
###### h6

h1
===
h2
___ this is underline
```

## Block quote

```markup
> this line will be highlighted
```

## Link

```markup
[link title](http://xxx.xxx.com/)

image from website
![image title](http://xxx.xxx.com/xxx/xxx.png)

image form local relative path
![image title](/xxx/xxx.png)
```

## List

```markup
* t1
* t2

+ t1
+ t2

- t1
    > block1
- t2
    > block2

1.  t1

    t1.1
2.  t2

    t2.1
```

## Separate Line

```markup
***
---
```

## Emphasize

```markup
*italics*
_italics_


**bold**
__bold__
```

## Code block

```markup
 \```language
 content here...

 \```
```

## Table

```markup
header|item1|item2
:-|:-:|-:
left|middle|right
```

## Special characters

```markup
\\ -> \
\` -> `
\* -> *
\_ -> _
\{\} -> {}
\[\] -> []
\(\) -> ()
\# -> #
\+ -> +
\- -> -
\. -> .
\! -> !
```

## Font color

```markup
<font color='#ff0000'>font</font>
```
